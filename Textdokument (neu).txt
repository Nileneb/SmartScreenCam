:: Miniconda herunterladen und installieren
curl -o Miniconda3-latest-Windows-x86_64.exe https://repo.anaconda.com/miniconda/Miniconda3-latest-Windows-x86_64.exe
start /wait Miniconda3-latest-Windows-x86_64.exe /InstallationType=AllUsers /S /D=C:\Miniconda3
C:\Miniconda3\Scripts\conda.exe init

:: CMD-Fenster neu starten

:: Git herunterladen und installieren
curl -o Git-2.41.0-64-bit.exe https://github.com/git-for-windows/git/releases/download/v2.41.0.windows.1/Git-2.41.0-64-bit.exe
start /wait Git-2.41.0-64-bit.exe /VERYSILENT /NORESTART

:: Repository klonen
git clone https://github.com/yourusername/smartscreencam.git
cd smartscreencam

:: Conda-Umgebung erstellen und aktivieren
conda env create -f env.yaml
conda activate smartscreencam_env

:: Falls notwendig, Abhängigkeiten installieren
pip install -r requirements.txt

:: Skript ausführen
python app.py
